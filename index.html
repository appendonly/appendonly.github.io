AppendOnly.io

Principled modern scalable datastore, there is no delete. Streaming, immutable, decentralised and easy to use. Released for free under the MIT License.

  • Learn more

Immutable Data

Once appended, data is not changed anymore. You can replay the stream and append more data, but you cannot ever change an entry.

const AO = require('appendonly')('YOUR-API-KEY')
// you can use any object as log identifier
const log = new AO({log: 123})
log.append(initialData)
   .then(log.append(nextItem))
   .then(x => console.log(x.mhash))

Content Addressed

Each entry has a multihash encoded hash-value. You must specify the hashes of your parent entries if you want to append to the log.
